# Product: Gwen

## Vision

Gwen is an open-source framework for building persistent, emotionally intelligent AI companions that run entirely on local hardware. It is a multi-model orchestration system that gives a local AI persistent identity, long-term memory, real-time voice, integrated life-coaching, and proactive initiative.

## What Gwen IS

- A **framework**, not a chatbot — the architecture is the product, personality is a loadable module
- A **local-first** system — all data stays on the user's hardware, zero cloud dependency
- A **multi-model orchestrator** — 3 logical tiers (Nerve, Voice, Deep Mind) mapped dynamically to physical models
- A **companion with memory** — 5-tier Living Memory (Stream, Chronicle, Map, Pulse, Bond)
- A **companion with time awareness** — 7 temporal senses, from circadian awareness to anniversary detection
- A **companion with emotional intelligence** — Amygdala Layer modulates storage, retrieval, and decay
- A **companion with coaching skills** — The Compass Framework (4 directions, 20 skills)
- A **companion with initiative** — Autonomy Engine initiates contact based on patterns and predictions

## What Gwen is NOT

- Not a therapist — never diagnoses, never treats, never claims clinical authority
- Not a cloud service — zero network calls, zero telemetry, zero data collection
- Not engagement-optimized — the Growth Principle pushes users toward real-world connection
- Not a single personality — soul-agnostic framework supports any companion identity

## Audience

- Primary: People who want a persistent, emotionally aware AI companion on their own hardware
- Secondary: Developers building custom companion experiences on the framework
- Tertiary: Researchers studying human-AI relational dynamics

## Design Principles

1. **Local-only, no telemetry** — all data stays on user hardware
2. **Soul-agnostic** — personality is a loadable module; the framework is the product
3. **Informed consent with friction** — Immersion Mode requires explicit opt-in with typed confirmation
4. **Growth over dependency** — companion pushes toward real-world connection
5. **Honesty** — truthful about AI nature in Grounded Mode; handled at system level in Immersion Mode
6. **Safety is bedrock** — operates below personality, below modes, below user preferences; cannot be disabled
7. **Graceful degradation** — runs on phones to workstations via Adaptive Profile System
